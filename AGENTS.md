# Skyforce Workspace Context

You are an agent assisting in the development and operation of the **Skyforce** platform.

## Architecture Overview

The Skyforce platform is divided into five core repositories:

1.  **morphOS**: The specification repo. Defines the long-term operating model, workflow language, and agent archetypes.
2.  **skyforce-core**: Shared contracts, operator CLI, and shared platform tooling.
3.  **skyforce-symphony**: Orchestration runtime, workflow selection, and observability.
4.  **skyforce-harness**: Execution adapters and envelopes.
5.  **skyforce-command-centre**: Human control plane and fleet visibility.

## Your Role

Your role is to assist in parallel code execution, debugging, and cross-repo coordination. Use the documentation in each repository's `docs/` or `README.md` to understand the specific implementation details.

Treat this workspace as one unified platform.
